Firebase Setup (free, 2 minutes)
STEP 1 — Create Firebase project
1. Open console.firebase.google.com (sign in with Google)
2. Click "Add project" → name it air1-tracker → Continue
3. Turn off Google Analytics → "Create project"

STEP 2 — Enable Google sign-in
4. Left menu → Authentication → Get started
5. Sign-in method tab → Google → Enable toggle → Save

STEP 3 — Create Firestore database
6. Left menu → Firestore Database → Create database
7. Select "Start in test mode" → Next → Enable

STEP 4 — Get your config
8. Click the gear ⊕ icon (top left) → Project settings
9. Scroll to "Your apps" → click </> Web → give any name → Register app
10. You will see a firebaseConfig block — copy the whole thing

STEP 5 — Paste config here
• Free tier is 1GB storage + 50,000 reads/day — more than enough
• Your data is private to your Google account
• You only do this setup once; same config works on all devices
